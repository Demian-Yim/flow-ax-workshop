# FLOW~ AX Quick Start — 패키지 개요

> "우리 조직에 AI 전환이 필요한지, 어디서 시작할지를 하루 만에 파악합니다"

---

## 1. 왜 Quick Start인가

**문제:** AI 도입을 검토하고 있지만, 어디서 시작해야 할지 모른다.
- "AI가 중요하다는 건 아는데, 우리 업무에 어떻게 적용하지?"
- "ChatGPT 써보라고 했는데, 다들 안 쓰고 있어"
- "컨설팅은 비싸고, 교육은 끝나면 잊어버려"

**해결:** 하루 만에 조직의 AI 전환 출발점을 잡는다.
- 참가자가 직접 자기 업무를 분석하고 우선순위를 도출
- 강의가 아닌 실습 — 끝나면 바로 쓸 수 있는 결과물이 남는다
- 원하면 SME/Mid-Corp으로 확장 가능한 첫 걸음

---

## 2. 3가지 옵션

| 옵션 | 시간 | 핵심 내용 | 산출물 | 가격 |
|------|------|----------|--------|------|
| **A. 반일 브리핑** | 3시간 | AI 전환 트렌드 + 조직 진단 미니 | 진단 요약 | 300만원 |
| **B. 1일 워크숍** | 7시간 | 브리핑 + RTC + ICEP 실습 | 캔버스 + 우선순위 | 500만원 |
| **C. 1일 + 리포트** | 7시간 + α | 워크숍 + 맞춤 진단 리포트 | 캔버스 + 리포트 15p | 800만원 |

**추천:** 옵션 B (1일 워크숍)가 가장 균형 잡힌 선택

---

## 3. 참가자 구성

| 항목 | 권장 |
|------|------|
| **인원** | 15~25명 (최소 10명, 최대 30명) |
| **구성** | 부서 혼합 권장 (다양한 관점 확보) |
| **직급** | 실무자~팀장급 (의사결정권자 1명 이상 참관 권장) |
| **사전 준비** | 없음 (사전 설문 15분만 응답) |

---

## 4. 참가자가 가져가는 것

### 개인
- 내 역할/업무/역량의 AI 전환 캔버스 (RTC 미니)
- AI 적용 우선순위 리스트 (ICEP 점수 기반)
- 1개 업무의 AI-Native 재설계 초안

### 조직
- 부서별 AI 전환 기회 맵
- 우선순위 Top 5 과제 목록
- 다음 단계 로드맵 (SME/Mid-Corp 확장 시)

---

## 5. 사전 준비 & 로지스틱스

### 고객사 준비 사항
- [ ] 워크숍 장소 확보 (테이블 토론 가능, 프로젝터)
- [ ] 참가자 확정 및 사전 설문 배포 (FLOW~ 제공)
- [ ] 노트북 또는 태블릿 지참 안내 (AI 도구 핸즈온용)
- [ ] Wi-Fi 확인

### FLOW~ 제공 항목
- [ ] 사전 설문 링크 (온라인, 15분)
- [ ] 워크숍 슬라이드 (투사용)
- [ ] 참가자 워크북 (PDF 또는 인쇄본)
- [ ] RTC 캔버스 / ICEP 매트릭스 템플릿
- [ ] 포스트잇, 마커 등 퍼실리테이션 도구

## 6. Quick Start → 확장 경로

```
Quick Start (1일)
    ↓ "더 깊이 하고 싶다"
SME Workshop (4~6주)
    ↓ "전사로 확대하고 싶다"
Mid-Corp AX (2~4개월)
    ↓ "거버넌스까지 구축하고 싶다"
Enterprise AX (4~8개월)
```

Quick Start는 **맛보기**가 아니라 **첫 번째 실행**입니다.
워크숍에서 도출한 결과물은 다음 단계의 입력값이 됩니다.
