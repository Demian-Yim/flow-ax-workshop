# FLOW~ AX Quick Start — 파일럿 영업 키트

> 목표: 2주 내 무료/저가 파일럿 **1건** 확보
> 대상: Quick Start 옵션 B (1일 워크숍, 15~25명)

---

## 1. 파일럿 제안 조건

| 항목 | 내용 |
|------|------|
| **형태** | 1일 워크숍 (7시간) |
| **참가 인원** | 15~25명 |
| **비용** | 무료 ~ 100만원 (정가 500만원의 80% 할인) |
| **할인 조건** | 케이스스터디 활용 동의 + 참가자 피드백 설문 협조 |
| **FLOW~ 제공** | 슬라이드 + 워크북 + 퍼실리테이션 + 진단 리포트(요약) |
| **고객사 제공** | 장소 + 프로젝터 + Wi-Fi + 참가자 확정 |

## 3. 컨택 메시지 템플릿

### 템플릿 A: 기존 고객 (카톡/이메일)

```
OO님, 안녕하세요. 임정훈입니다.

요즘 AI 전환(AX) 관련 워크숍 프로그램을 개발하고 있습니다.
"직원들이 자기 업무를 직접 분석하고, AI 적용 우선순위를 도출하는" 
1일짜리 실습형 워크숍인데요.

베타 테스트 차원에서 1~2개 기업에 무료로 진행하려 합니다.
혹시 관심 있으시거나, 주변에 적합한 곳이 있으면 소개 부탁드려도 될까요?

- 형태: 1일 워크숍 (7시간)
- 인원: 15~25명
- 비용: 무료 (피드백 설문 협조)
- 일정: 4~5월 중 조율

간단한 소개 자료 보내드릴까요?
```

혹시 팀 단위로 한번 체험해보시겠어요?

## 4. 첨부용 1페이지 소개 자료

**파일명:** `FLOW_AX_QuickStart_1pager.md` (별도 작성 권장)

**구성:**
1. 한 줄 소개: "AI 전환, 어디서 시작할 것인가 — 1일 만에 답을 찾습니다"
2. 워크숍 흐름 (5블록 시각화)
3. 참가자가 가져가는 것 (3가지)
4. Quick Start → SME → Mid-Corp 확장 경로
5. 파일럿 조건 (무료, 피드백 협조)
6. 연락처

## 5. 파일럿 성공 후 수확 계획

### 워크숍 당일
- [ ] 사진 촬영 (참가자 동의 하에)
- [ ] 참가자 피드백 설문 (QR)
- [ ] 의사결정권자와 15분 1:1 대화 ("다음 단계 관심 있으신가요?")

### 워크숍 후 3일 이내
- [ ] 감사 이메일 + 워크숍 결과 요약 리포트 발송
- [ ] 만족도 수치 정리 (케이스스터디용)
- [ ] "SME Workshop 제안서 보내드릴까요?" 후속 질문

### 워크숍 후 1주 이내
- [ ] 케이스스터디 1페이지 작성
- [ ] LinkedIn 후기 포스트 발행 (고객사 동의 시)
- [ ] Master Proposal에 사례 반영
- [ ] 추천 요청: "비슷한 고민을 가진 다른 분 소개해주실 수 있을까요?"

### 30일 후
- [ ] 참가자 30일 체크인 (버디 시스템 후속)
- [ ] EARS 지표 변화 확인
- [ ] 2차 워크숍 or SME 확장 제안
