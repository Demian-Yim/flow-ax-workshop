### 10:45-12:15 | 블록 2: RTC 캔버스 실습 (90분)

**목표:** 내 역할·업무·역량을 한 장에 가시화

| 시간 | 활동 | 방법 |
|------|------|------|
| 10:45-10:55 | RTC 캔버스 소개 | R(역할) + T(업무) + C(역량) 구조 설명 |
| 10:55-11:05 | 사례 시연 | 가상 사례로 RTC 캔버스 작성 과정 시연 |
| 11:05-11:45 | **개인 실습** | 각자 자기 RTC 캔버스 작성 (40분) |
| | | → R영역: 내 핵심 역할 3가지 |
| | | → T영역: 주요 업무 + 시간 배분 + AI 적합도 표시 |
| | | → C영역: 현재 역량 vs AI 협업 시 필요 역량 |
| 11:45-12:05 | **테이블 공유** | 4~5인 그룹에서 서로의 캔버스 피드백 |
| 12:05-12:15 | 전체 정리 | "가장 놀라웠던 발견" 2~3명 공유 |

**준비물:** RTC 캔버스 워크시트 (A3 인쇄 or PDF), 슬라이드 #16~#22
