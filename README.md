# FLOW~ AX Workshop Platform
### 플로우 AX 워크샵 플랫폼

> AI와 함께 일의 흐름을 디자인하는 실전 워크샵 플랫폼

## 🌊 소개
AI 해커톤, 바이브코딩, AX 직무 리디자인을 위한 올인원 워크샵 플랫폼입니다.

## ✨ 핵심 기능
- 🤖 **AI 해커톤** — 5단계 스프린트 (이해→스케치→결정→프로토타입→테스트)
- 🎵 **바이브코딩** — 4단계 스프린트 (아이디어→프롬프트→빌드→공유)
- 🔄 **AX 직무 리디자인** — 4단계 스프린트 (AS-IS→AI매핑→TO-BE→실행)

## 🚀 시작하기
```bash
# 로컬 서버 실행
npx live-server --port=3000

# 또는
npm run dev
```

## 📱 페이지 구조
| 페이지 | URL | 설명 |
|--------|-----|------|
| 랜딩 | `/` | 참가자 코드 입장 |
| 관리자 | `/admin` | 워크샵 관리, 팀 모니터링 |
| 대시보드 | `/dashboard` | 참가자 스프린트 진행 |
| 갤러리 | `/gallery` | 명예의 전당, 투표 |
