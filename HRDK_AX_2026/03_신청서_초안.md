## Ⅱ. AI 솔루션 소개

### 1. 솔루션명: 제안서 작성 AI Agent (가칭 **PropAI**)

### 2. 핵심 기능
1. 발주처 RFP·고객 정보 입력 → 맞춤 제안서 초안 5분 내 생성
2. 회사 프로필·실적·차별화 포인트 자동 삽입
3. 톤·길이·강조점 조정 (영업/기술/관리 등 직무별 최적화)
4. 협업 코멘트·버전관리

### 3. 기술 스택
- LLM(Claude/Gemini) + RAG (회사 내부 자료 색인)
- Next.js + Firebase / Vercel 배포
- 사용자 인증·과금 (Firebase Auth + Stripe)

### 4. 도입 효과 (정량 가설)
- 제안서 초안 작성 시간 **8시간 → 1시간** (87.5% ↓)
- 영업·기획 직군 1인당 월 **20시간 절감**
- 제안서 품질 표준화 → 수주율 향상

### 5. 차별성
- **HRD 전문기관 개발** → "사용 교육"이 솔루션의 일부로 설계됨
- 영업·기획 외에 **HR·기획·운영 등 모든 문서 직무 확장 가능**
