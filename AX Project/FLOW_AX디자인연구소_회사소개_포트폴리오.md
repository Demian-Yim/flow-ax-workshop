HRD 15년 현장 경험을 기반으로, AI 리터러시 교육에서 업무 재설계, 파일럿 실행, 성과 측정까지 — 전환의 전 과정을 교육과 코칭으로 설계합니다.

기술 컨설턴트가 도입하고 떠난 자리에서, FLOW~는 사람이 전환을 완수할 때까지 현장에 있습니다.

### 우리가 하는 일 (서비스 영역)

FLOW~ : AX Design Lab(플로우~ : AX 디자인 연구소)은 네 가지 방식으로 기업의 AI 전환을 설계합니다.

**교육 (Teaching & Training)**
AI 리터러시 기본반·실무반, AI 협업 역량 워크숍, 직군별 AI 활용 교육.
"모르는 것을 아는 것으로" 바꾸는 첫 단계를 설계합니다.

**퍼실리테이션 (Facilitation)**
RTC 캔버스 워크숍, ICEP 스코어링 세션, WHY-WHAT-HOW 업무 재설계 워크숍.
구성원이 스스로 답을 찾도록 구조화된 대화를 이끕니다.

**코칭 (Coaching)**
STAR-AR 변화관리 코칭, AX Champion 육성, 저항 유형별 현장 코칭.
전환 과정에서 멈춘 사람 옆에 서서, 다음 한 걸음을 함께 설계합니다.

**컨설팅 (Consulting)**
AX 진단, Phase 0-5 전체 설계, 파일럿 운영, EARS 성과 측정, 거버넌스 수립.
전환의 전체 그림을 그리고, 실행까지 책임집니다.

## 3. 고유 방법론 자산

FLOW~ : AX Design Lab(플로우~ : AX 디자인 연구소)은 현장에서 검증한 6개의 고유 프레임워크를 보유하고 있습니다.

### 프레임워크 일람

| 프레임워크 | 용도 | 적용 Phase |
|---|---|---|
| RTC 캔버스 | 역할(R)·업무(T)·역량(C) 통합 가시화 | Phase 1 |
| ICEP 매트릭스 | AI 적용 우선순위 결정 (사람 준비도 포함) | Phase 2 |
| WHY-WHAT-HOW | 업무 재설계 워크숍 코칭 프레임 | Phase 3 |
| STAR-AR | 변화관리 코칭 (진단→과제→행동→결과→대안→성찰) | Phase 4 |
| EARS | 성과 측정 (효과성·정렬도·적합도·지속가능성) | Phase 5 |
| ARQ | 품질 검증 (정확도·적합도·품질) | Phase 5 |

### 방법론 차별점

일반적인 AX 컨설팅이 "업무 가시화 → ICE 우선순위 → 도입"으로 끝나는 데 반해,
FLOW~ AX는 "사람 준비(Phase 0) → 역할·역량 통합(Phase 1) → 사람 포함 우선순위(Phase 2) → 워크숍 기반 재설계(Phase 3) → 코칭 기반 변화관리(Phase 4) → 3중 성과 검증(Phase 5)"으로 사람의 전환 전 과정을 설계합니다.

## 4. 프로그램 포트폴리오

### 4-1. 강의 (Teaching & Lecture)

**AI 리터러시 기본반** (1일, 8h)
대상: 전 구성원
내용: AI란 무엇인가 → 주요 AI 도구 체험 → 미니 프로젝트 → 공유 및 토론
특징: 이론 30% + 실습 70%, 참가자가 직접 AI를 사용해보는 핸즈온 중심

**AI 리터러시 실무반** (1일, 8h)
대상: Phase 0 기본반 수료자
내용: 프롬프트 엔지니어링 심화 → 자기 업무에 AI 도구 매핑 → 워크플로우 설계 → 팀 협업 시나리오
특징: 참가자의 실제 업무를 소재로 사용, 교육 후 즉시 활용 가능한 결과물 산출

**AI 시대의 리더십** (2-4h)
대상: 팀장급 이상
내용: AI가 바꾸는 리더의 역할 → 팀 AI 전환 전략 → 저항 관리 → AX Champion의 조건
특징: 리더가 전환의 방관자가 아닌 설계자가 되도록 관점 전환

**직군별 AI 활용 교육** (2-4h, 커스텀)
대상: 마케팅 / 재무 / 인사 / 영업 / 고객지원 / R&D
내용: 직군의 핵심 업무 → AI 적용 가능 업무 식별 → As-Is → To-Be 시나리오 → 실습
특징: 직군별 맞춤 시나리오, 실무자가 "내 일에 쓸 수 있겠다"고 느끼는 순간 설계


### 4-2. 퍼실리테이션 (Facilitation)

**RTC 캔버스 워크숍** (3-4h)
대상: 팀 단위 (5-15명)
내용: 역할(R) 정의 → 업무(T) 가시화 → 역량(C) 진단 → AI 적합도 판정 → 조직 업무 지도 완성
산출물: 팀별 RTC 캔버스, AI 적용 후보 업무 리스트, 역량 갭 리포트

**ICEP 스코어링 세션** (2-3h)
대상: 의사결정자 + 실무 대표
내용: RTC 캔버스 기반 후보 업무 리뷰 → ICEP 4축 평가 → Wave 분류 → 파일럿 대상 확정
산출물: ICEP 스코어카드, Wave별 실행 로드맵

**WHY-WHAT-HOW 업무 재설계 워크숍** (1일, 8h)
대상: 파일럿 대상 팀
내용: WHY(2h) 왜 바꾸는가 → WHAT(3h) 무엇을 바꾸는가 → HOW(3h) 어떻게 실행하는가
산출물: As-Is → To-Be 전환 시나리오, AI 도구 매핑, 프롬프트 초안, 핸드오프 정의


### 4-3. 코칭 (Coaching)

**STAR-AR 전환 코칭** (1:1 또는 소그룹, 회당 1-2h)
대상: 전환 과정에서 어려움을 겪는 구성원, 저항 유형별 대상자
내용: 상황 진단(S) → 과제 설정(T) → 행동 설계(A) → 결과 확인(R) → 대안 탐색(A) → 성찰(R)
특징: 저항 4유형(두려움/무능/관성/불신)별 맞춤 접근, 단회성이 아닌 과정 코칭

**AX Champion 육성 프로그램** (2주, 집중 과정)
대상: 팀별 AI 전환 리더 후보 (3축 선발: AI 친화도 + 동료 영향력 + 학습 의지)
내용: AI 심화 교육 → 퍼실리테이션 기법 → 변화관리 기초 → 팀 내 전파 실습 → 역할 정의
산출물: Champion 활동 계획서, 팀 전환 로드맵

**리더 전환 코칭** (1:1, 월 2회)
대상: 팀장·임원급
내용: AI 전환에서 리더의 역할 재정의, 팀 저항 대응 전략, 의사결정 지원
특징: 리더가 "시키는 사람"이 아니라 "함께 전환하는 사람"이 되는 과정 설계


### 4-4. 컨설팅 (Consulting)

**AX 진단 및 로드맵 설계** (2-4주)
범위: ICEP 간이 진단 → Phase 0-5 커스텀 설계 → 일정·예산·성과 기준 수립
산출물: AX 진단 리포트, Phase별 실행 계획서, KPI 정의서

**파일럿 운영 지원** (4-6주)
범위: 파일럿 팀 선정 → 환경 세팅 → 주간 코칭 → 중간 점검 → 성과 평가
산출물: 파일럿 결과 리포트, 확산 여부 판단 데이터, 개선 권고안

**EARS 성과 측정 및 거버넌스 수립** (지속)
범위: 효과성·정렬도·적합도·지속가능성 4축 측정 → 3 Wave 확산 설계 → 거버넌스 정책 수립
산출물: EARS 대시보드, 거버넌스 가이드라인, Wave별 확산 계획

## 5. FLOW~ AX 방법론 전체 구조

Phase 0  AI 리터러시 선행 교육 (2주)
↓
Phase 1  역할·업무·역량 가시화 — RTC 캔버스 (2-3주)
↓
Phase 2  AI 적용 우선순위 결정 — ICEP 매트릭스 (1-2주)
↓
Phase 3  AI-Native 업무 재설계 — WHY-WHAT-HOW 워크숍 (2-4주)
↓
Phase 4  파일럿 실행 + 변화관리 — STAR-AR 코칭 (4-6주)
↓
Phase 5  측정·확산·거버넌스 — EARS + ARQ (지속)
↺ 피드백 루프
